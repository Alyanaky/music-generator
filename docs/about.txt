Полное описание проекта "Генератор музыки"
Функционал
Генерация мелодий:

Создание случайных мелодий на основе заданных параметров (например, тональность, длина мелодии).
Использование алгоритмов марковских цепей для генерации мелодий.
Генерация аккордов:

Создание аккордовых прогрессий на основе заданной тональности.
Поддержка различных типов аккордов (мажорные, минорные, септаккорды и т.д.).
Генерация ритмов:

Создание ритмических паттернов с использованием L-систем или других алгоритмов.
Поддержка различных ритмических структур (например, 4/4, 3/4).
Экспорт музыки:

Экспорт генерированной музыки в формате MIDI или WAV.
Возможность сохранения файлов в указанную пользователем директорию.
Взаимодействие с пользователем
Загрузка аудиофайлов:

Пользователь может загрузить существующие аудиофайлы для анализа и визуализации.
Поддержка форматов WAV и MP3.
Настройка параметров генерации:

Пользователь может задавать параметры для генерации мелодий, аккордов и ритмов (например, тональность, длина мелодии, тип ритма).
Визуализация:

Отображение волновой формы загруженных аудиофайлов.
Отображение спектрограмм и других визуальных представлений аудиоданных.
Поддерживаемые файлы
Аудиофайлы: WAV, MP3.
Экспорт: MIDI, WAV.
Вид интерфейса
Основное окно:

Кнопка "Загрузить аудиофайл" для выбора и загрузки аудиофайла.
Кнопка "Сгенерировать мелодию" для создания случайной мелодии.
Кнопка "Сгенерировать аккорды" для создания аккордовой прогрессии.
Кнопка "Сгенерировать ритм" для создания ритмического паттерна.
Холст для отображения волновой формы и спектрограмм.
Панель настроек:

Поля для ввода параметров генерации (например, тональность, длина мелодии).
Кнопка "Экспортировать" для сохранения генерированной музыки в файл.
Действия кнопок
Загрузить аудиофайл:

Открывает диалоговое окно для выбора аудиофайла.
Загружает выбранный файл и отображает его волновую форму на холсте.
Сгенерировать мелодию:

Генерирует случайную мелодию на основе заданных параметров.
Отображает волновую форму или спектрограмму генерированной мелодии на холсте.
Сгенерировать аккорды:

Генерирует аккордовую прогрессию на основе заданных параметров.
Отображает волновую форму или спектрограмму генерированных аккордов на холсте.
Сгенерировать ритм:

Генерирует ритмический паттерн на основе заданных параметров.
Отображает волновую форму или спектрограмму генерированного ритма на холсте.
Экспортировать:

Открывает диалоговое окно для выбора директории сохранения.
Сохраняет генерированную музыку в указанную директорию в формате MIDI или WAV.

Этот проект предоставляет пользователю возможность загружать и анализировать аудиофайлы, генерировать мелодии, аккорды и ритмы, а также экспортировать генерированную музыку. Интерфейс интуитивно понятен и позволяет пользователю легко взаимодействовать с программой. Удачи в разработке!
